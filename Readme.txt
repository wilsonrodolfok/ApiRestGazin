- Para criação da API foi utilizado o .NET CORE 5, codificados no Microsoft Visual Studio 2022 Community;
- Utilizados pacotes de framework Entity Framework (Core, tools e SqlServer); Utilizado também Swagger para consumo da API;
- O Banco de dados foi utilizado SqlServerExpress(Instalado de forma padrão e com autenticação padrão para fins de desenvolvimento do teste);

- A codificação foi feita utilizado CODE FIRST e em seguida utilizado o MIGRATIONS para criação do banco de dados no SQLServer;
- Tentei fazer a implementação de acordo com os padrões de desenvolvimento (herança, reuso, factory, injeção de dependencias, clean code, etc);
- Os CRUDS estão disponíveis para consumo, para teste compilar o projeto dentro do VisualStudio2022;
- Não consegui deixar pronto o FrontEnd;
